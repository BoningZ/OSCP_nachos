Contents:

  Sample course handouts for undergraduate operating system course using Nachos.

Generic:

    course.pdf      course syllabus, and assignment dates

    overview.pdf    overview of Nachos project

    mechanics.pdf   mechanics of finding the code and turning in assignments
                    (will vary from school to school)

    nethelp.pdf     overview of how network protocols work (a weak area
                    in many OS textbooks)

    reading.txt     sample reading list, to supplement standard textbooks


Sample assignments:

    thread.pdf      concurrency and threads

    userprog.pdf    multiprogramming and system calls

    vm.pdf          virtual memory and TLB handling

    filesys.pdf     file system

    network.pdf     network protocol and distributed system
